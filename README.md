# Portfolio – Lucas Antunes

Site estático para portfólio, com cards gerados via `data/projects.json`.

## Como rodar local
Abra `index.html` no navegador (ou use uma extensão de Live Server do VS Code).

## Adicionar projeto
1. Imagem: `assets/img/projects/<slug>/cover.jpg`
2. Editar `data/projects.json` e incluir o objeto do projeto.
3. Commit & push.

## Deploy
Automático no GitHub Pages via GitHub Actions (branch `main`).
